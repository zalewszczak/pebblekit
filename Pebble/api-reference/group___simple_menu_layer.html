SimpleMenuLayer
Layers

Detailed Description

Wrapper around MenuLayer, that uses static data to display a list menu.

struct SimpleMenuLayer

Data structure of a SimpleMenuLayer.

Note
a SimpleMenuLayer * can safely be casted to a Layer * and to a MenuLayer * can thus be used with other all functions that take a Layer * or MenuLayer *, respectively, as argument.
For example, the following is legal:
SimpleMenuLayer simple_menu_layer;
...
layer_set_hidden((Layer *)&simple_menu_layer, true);
However there are a few caveats:
  • Do not try to change to bounds or frame of a simple menu layer, after initializing it.
