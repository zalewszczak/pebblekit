InverterLayer
Layers

Detailed Description

Layer that inverters anything "below it".

This layer takes what has been drawn into the graphics context by layers that are "behind" it in the layer hierarchy. Then, the inverter layer uses its geometric information (bounds, frame) as the area to invert in the graphics context. Inverting will cause black pixels to become white and vice versa.

The InverterLayer is useful, for example to highlight the selected item in a menu. In face, the MenuLayer itself uses InverterLayer to accomplish it selection highlighting.

Note
a InverterLayer * can safely be casted to a Layer * and can thus be used with other all functions that take a Layer * as argument.
For example, the following is legal:
InverterLayer inverter_layer;
...
layer_set_frame((Layer *)&inverter_layer, GRect(10, 10, 50, 50));
