Media Utilities
Foundation

Detailed Description

Utilities to help using media resources.

HeapBitmap

The function of HeapBitmap is to load a GBitmap from a resource pack into memory so that it can be used with a BitmapLayer, or using the lower-level graphics_draw_bitmap_in_rect() bitmap drawing function inside a custom Layer's .update_proc.

BmpContainer

The BmpContainer is a composite of BitmapLayer and a heap-backed image resource, which is provided for convenience and to reduce boiler plate code of loading a bitmap into memory and displaying it using a BitmapLayer.
