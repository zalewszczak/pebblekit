Data Structure Documentation

struct NumberWindowCallbacks

Data structure containing all the callbacks for a NumberWindow.

Data Fields
NumberWindowCallback decremented Callback that gets called as the value is decremented. Optional, leave NULL if unused.
NumberWindowCallback incremented Callback that gets called as the value is incremented. Optional, leave NULL if unused.
NumberWindowCallback selected Callback that gets called as the value is confirmed, in other words the SELECT button is clicked. Optional, leave NULL if unused.
struct NumberWindow

Data structure of a NumberWindow.

Note
a NumberWindow * can safely be casted to a Window * and can thus be used with other all functions that take a Window * as argument.
For example, the following is legal:
NumberWindow number_window;
...
window_stack_push((Window *)&number_window, true);
